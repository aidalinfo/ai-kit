Cette section documente les classes et fonctions publiques du package `@ai_kit/core`. Les pages suivantes détaillent les constructeurs, options et méthodes d’`Agent`, `Workflow`, `WorkflowRun` et `RuntimeStore`.

## Conventions

- Les types proviennent du SDK `ai` lorsque c’est pertinent (`LanguageModel`, `ToolSet`, etc.).
- Les signatures sont exprimées en TypeScript ; adaptez-les selon votre langage cible.
- Les options marquées comme *optionnel* disposent de valeurs par défaut explicitées dans la documentation.
